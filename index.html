Why Pulse UI

Everything you need,
nothing you don't

Designed for developers and designers who ship fast. Every component is production-ready and dark-mode native.

Design tokens
Every color, radius, spacing, and type scale lives as a CSS variable. Swap your brand in seconds.
React-ready
Drop-in React components with sensible props. Works with Tailwind, Vite, Next.js, or vanilla.
Dark mode native
Every token ships with a dark variant. Automatic via prefers-color-scheme. Zero extra work.
Accessible
WCAG 2.1 AA contrast ratios throughout. Focus rings, aria attributes, and keyboard nav built in.
Semantic HTML
Clean, minimal markup. Use the CSS classes standalone, or import the React wrapper components.
MIT license
Use it on client projects, sell products built with it. No attribution required. Truly free.
